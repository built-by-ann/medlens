# Roadmap

This roadmap tracks MedLens's development milestones across four sprints, from backend foundation through production engineering.

---

## Sprint 1 (Completed)

**Goal:** Set up the project structure, backend foundation, database, Docker environment, and authentication.

- Project setup
- FastAPI
- Docker
- PostgreSQL
- SQLAlchemy
- Alembic
- Authentication (registration and login)
- JWT
- `/users/me`
- Health endpoint
- Backend tests
- Documentation

---
